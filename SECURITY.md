# Security and Privacy

This repository is a public portfolio lab. It contains only synthetic data.

It does not contain:
- employer data
- real tenant IDs
- real usernames or UPNs
- real device IDs
- real serial numbers
- real asset tags
- real tickets
- real BitLocker recovery keys
- hardware hashes
- app registration secrets
- screenshots from production systems

Any Microsoft Graph or Intune examples are either dry-run, mocked, or intended only for a disposable development tenant with fake users and fake devices.

## Reporting a concern

Open an issue if you find sample data that looks private or unsafe. Do not paste secrets or private data into the issue. Describe the file path and the pattern that looks risky.
