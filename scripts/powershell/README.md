# PowerShell Scripts

These scripts operate on synthetic data by default.

- `Invoke-EndpointReadinessCheck.ps1`: validates endpoint readiness fields.
- `New-EndpointReadinessReport.ps1`: writes the Markdown readiness report.
- `New-PatchRiskReport.ps1`: writes the Markdown patch-risk report.
- `Invoke-GraphDryRun.ps1`: reads mock Graph responses only.

No script requires a real Microsoft tenant for the MVP.
